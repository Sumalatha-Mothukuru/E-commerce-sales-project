## 📊 Project Overview  

This project focuses on analyzing E-Commerce sales data using Microsoft Excel to derive meaningful business insights.  

The dashboard provides a comprehensive view of revenue, profit, order volume, product performance, regional contribution, and payment behavior.  

The primary objective of this project is to monitor key performance indicators (KPIs), identify growth opportunities, and support data-driven business decisions through interactive visualizations.

---

## 🛠️ Key Skills & Tools Used  

- Microsoft Excel  
- Pivot Tables & Pivot Charts  
- Data Cleaning & Data Transformation  
- KPI Design & Performance Tracking  
- Slicers & Interactive Filtering  
- Dashboard Layout & Visualization Design  
- Business Data Analysis  

---

## 📈 Trend & Performance Analysis  

- Analyzed monthly sales trends to identify peak revenue periods.  
- Evaluated category-wise profit performance.  
- Compared regional sales contribution.  
- Studied payment mode distribution impact on revenue.  
- Examined sales vs profit relationship to understand margin patterns.  

---

## 📌 Key Findings & Insights  

- Certain product categories contribute higher profit margins despite moderate sales.  
- Regional performance varies significantly, indicating potential growth areas.  
- Monthly trend analysis highlights seasonal sales patterns.  
- Payment mode analysis shows balanced customer preference distribution.  
- Sales growth does not always directly translate to proportional profit growth.
